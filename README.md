# Kanamaneni Rohith - Portfolio

A responsive, framework-free personal portfolio built with semantic HTML, modern CSS, and ES6 modules.

## Run locally

Open `index.html` in a modern browser, or serve the directory with any static file server.

## Before deploying

- Update `sitemap.xml` with your deployed domain.
- Add your resume PDF under `assets/resume/` and link it from the hero if desired.
